Editable reports

Allows to edit report data online
-This module requires the installation of PhantomJS
-After installation, add a parameter with key=addons_path and the absolute path of project_addons folder

This module only works with views that have configured these parameters:
- View type: qweb
- Object: A model from which the view will render its data
- External ID: A unique identifier generated from the view form

Inside view qweb code, global variable "record_id" refers to active model record
All the forms need the class "report_form" to be processed.

View code example:

                    
<form id="client-data" role="form" method="post" class="report_form" t-att-model="record_id._name" t-att-record="record_id.id">
    <h2>CLIENT DATA</h2>
    <p>Name:
        <input class="form-control" name="name" t-att-value="record_id.name"/>
    </p>
    <p>Phone:
        <input class="form-control" name="phone" t-att-value="record_id.phone"/>
    </p>
    <p>e-mail:
        <input class="form-control" name="email" t-att-value="record_id.email"/>
    </p>
</form>
